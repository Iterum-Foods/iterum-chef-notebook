Iterum R&D Chef Notebook - Quickstart
====================================

**How to Launch the App:**

1. Double-click `start_here.bat` in this folder.
2. Wait a few moments while the app sets up and launches.
3. Your browser will open automatically to the app's main page.

**What You Need:**
- Windows PC
- Python 3.9 or newer (https://www.python.org/downloads/)

**What Happens:**
- The app will check for Python and required packages.
- It will install any missing dependencies automatically.
- Both the backend (API) and frontend (web app) will start.
- The app will open in your browser at http://localhost:8080

**To Stop the App:**
- Click in the black command window and press Ctrl+C

**Troubleshooting:**
- If you see an error about Python or pip, install Python from https://www.python.org/downloads/
- If you see a firewall prompt, allow access so the app can run locally.
- If you have issues, close all app windows and try again.

**Feedback:**
- Please report any bugs or suggestions to the developer!

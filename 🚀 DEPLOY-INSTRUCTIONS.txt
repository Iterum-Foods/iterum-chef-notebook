OPTION 2: MANUAL SETUP (STEP-BY-STEP)
--------------------------------------
1. Open PowerShell in this directory

2. Login to Firebase:
   > firebase login

3. Initialize hosting:
   > firebase init hosting
   
   Answer prompts:
   - Existing project: Yes
   - Project: usta-app
   - Public directory: usta-public
   - Single-page app: Yes
   - GitHub: No

4. Prepare files:
   > .\PREPARE-USTA-DEPLOYMENT.bat

5. Deploy:
   > firebase deploy --only hosting

6. Done! Check the URL shown in terminal

TROUBLESHOOTING:
----------------

Problem: "Not authorized"
Solution: firebase login --reauth

Problem: "No project found"
Solution: Create project at console.firebase.google.com first
